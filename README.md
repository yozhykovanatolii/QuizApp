# QuizApp

A simple mobile application that presents multiple-choice questions to the user, automatically moves to the next question after an answer is selected, and displays the final result at the end. The application also stores and shows the history of previous quiz results.

## Features

- Multiple-choice questions screen.
- Automatic transition to the next question after selecting an answer.
- Animated transitions between questions.
- Final results screen displaying the number of correct answers.
- Results history screen showing previous quiz attempts.
- Results history screen showing previous quiz attempts.

## Tech Stack

**Core Framework**
- **Flutter** – Cross-platform UI framework  
- **Dart** – Programming language  

**State Management**
- **Cubit**

**Dependency Injection**
- **GetIt** – Service locator and dependency injection

**Storage**
- **Shared Preferences** – Local data persistence

## Getting Started

### Prerequisites
- Flutter SDK (>= 3.x)
- Dart
- Android Studio or VS Code
- Android emulator or physical Android device

### Installation
1. Clone the repository
```
git clone https://github.com/yozhykovanatolii/QuizApp.git
```

2. Install dependencies
```
flutter pub get
```
3. Run the application
```
flutter run
```
